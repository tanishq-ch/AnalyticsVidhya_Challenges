## 🛠️ General Tech Stack

Most solutions in this repository are built using:

- **Language:** Python 3.8+
- **Core ML:** `scikit-learn`, `catboost`, `lightgbm`, `xgboost`
- **Data:** `pandas`, `numpy`
- **Validation:** K-Fold Cross-Validation, OOF ensembling
- **Experiment Tracking:** Inline comments & fold-wise logging

Each challenge folder contains its own `README.md` with a detailed breakdown of the approach, feature engineering, and results.
